Chat_bot
FAQ Chatbot using NLP

Overview
Simple FAQ chatbot built using NLP techniques like TF-IDF and cosine similarity.

Features
- Text preprocessing (stopword removal)
- TF-IDF vectorization
- Semantic similarity matching
- Command-line chatbot

Tools
- Python
- Pandas
- Scikit-learn
- NLTK

How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the chatbot:
   python app.py
